Smart Employee Performance & Attrition Prediction System
📌 Project Overview

This project is an end-to-end Machine Learning application that predicts:

🧑‍💼 Employee Attrition (whether an employee is likely to leave the company)
📊 Employee Performance Rating

It helps organizations make data-driven HR decisions by identifying employees at risk of leaving and understanding performance trends.

🎯 Objectives
Predict employee attrition using classification algorithms
Analyze factors affecting employee performance
Build a complete ML pipeline from data preprocessing to deployment
Deploy an interactive web app using Streamlit

Technologies Used:
Programming -	Python,
Data Handling -	Pandas, NumPy,
Visualization -	Matplotlib, Seaborn,
ML Models -	Scikit_learn,
Deployment - 	Streamlit,
Model Saving - 	Joblib,
